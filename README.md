**처음처럼 톡** 은 리눅스 데스크탑 기반의 오픈소스 채팅 웹 앱이다. "처음처럼" 초심을 잃지 않으려 노력하지만 결국 "처음처럼"으로 하루를 마무리하는 그대들의 회사에 원활한 소통을 위한 필수적인 채팅 앱이다. 

<처음처럼 톡>은 개발자 그룹 <처음이조>에서 개발되었다.
